# Curio Frontend

큐리오 프로젝트 프론트엔드입니다.  
React + Vite 기반으로 개발되었습니다.

## 프로젝트 설치

1. Node.js 설치 (v18 이상 권장)

2. 레포 클론
```bash
git clone https://github.com/CURIO-BSSM/curio-frontend
cd curio-frontend
```

## branch
dev에서 개발하고 푸시할 것

## 📌 커밋 컨벤션

| 타입 | 설명 |
|------|------|
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| chore | 빌드 설정, 개발환경 설정 등 |
| refactor | 코드 리팩토링 |
| style | 코드 스타일/포맷 변경 (기능 변화 없음) |
| docs | 문서 수정 |
| test | 테스트 코드 추가 |
| add | 파일 추가 |
